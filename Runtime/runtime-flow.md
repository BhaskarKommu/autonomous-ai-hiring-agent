# Runtime Execution Flow

## Request Lifecycle

1. User submits request
2. Planner analyzes intent
3. Skills engine selects workflows
4. Executors perform actions
5. Reflection engine evaluates results
6. Memory system stores learnings

---

## Example

Input:
"Find top AI backend engineers"

Execution:
- Detect hiring workflow
- Select semantic search skill
- Retrieve candidates
- Rank candidates
- Generate recommendations
- Evaluate workflow quality
- Persist recruiter preferences
