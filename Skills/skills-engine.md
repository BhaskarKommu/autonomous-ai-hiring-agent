# Skills Engine

Skills represent reusable execution workflows.

## Available Skills

### SemanticSearchSkill
Retrieves candidates using vector similarity.

### CandidateRankingSkill
Scores candidates based on relevance and recruiter preferences.

### RecommendationSkill
Generates intelligent hiring recommendations.

### ReflectionSkill
Evaluates workflow quality and improvement opportunities.

---

## Skill Lifecycle

1. Skill selected by planner
2. Skill executed by runtime
3. Results evaluated
4. Improvements stored in memory
