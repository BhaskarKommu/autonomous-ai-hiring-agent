# Autonomous AI Hiring Agent

A Hermes-inspired self-improving AI hiring runtime built with ASP.NET Core, semantic search, vector intelligence, memory systems, autonomous planning, and skill orchestration.

## Features

- Autonomous AI planning
- Semantic candidate search
- AI memory system
- Skill-based workflow execution
- Reflection and self-improvement loops
- Intelligent candidate ranking
- Vector search integration

## Architecture

User Request
→ Planner
→ Skill Engine
→ Tool Executors
→ Reflection Engine
→ Learning Memory

## Tech Stack

- ASP.NET Core
- PostgreSQL + pgvector
- OpenAI APIs
- SignalR
- Semantic Search
- AI Orchestration
