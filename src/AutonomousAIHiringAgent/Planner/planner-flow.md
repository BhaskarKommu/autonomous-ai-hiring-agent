# Planner Logic

## Responsibilities

- Analyze user intent
- Detect workflow type
- Select required skills
- Build execution sequence
- Coordinate runtime actions

---

## Workflow Example

Input:
"Find AI engineers with semantic search experience"

Planner Analysis:
- Domain: Hiring
- Required Action: Candidate Search
- Skills Needed:
  - SemanticSearchSkill
  - RankingSkill
  - RecommendationSkill

Execution Plan:
1. Run semantic retrieval
2. Apply ranking engine
3. Generate candidate recommendations
4. Trigger reflection
5. Save learning memory
