# Planner Engine

The planner analyzes user requests and creates execution workflows.

## Responsibilities

- Intent analysis
- Workflow generation
- Skill selection
- Tool orchestration
- Execution sequencing

## Example

Input:
"Find top AI backend engineers for fintech startup"

Planner Output:
1. Select semantic search skill
2. Run candidate retrieval
3. Apply ranking engine
4. Generate recommendations
5. Store execution memory
6. Trigger reflection loop
